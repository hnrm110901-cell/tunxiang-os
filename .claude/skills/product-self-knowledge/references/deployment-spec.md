## 网络拓扑

```
腾讯云 (FastAPI + PG)
    ↕ Tailscale
Mac mini M4 (本地PG + AI, port 8000/8100)
    ↕ WiFi局域网
安卓POS主机 / 安卓KDS平板 / 员工手机
    ↕ USB
打印机 / 电子秤 / 钱箱 / 扫码枪
```

**铁律：所有外设只连安卓POS主机，Mac mini不连任何外设。**

## 云端配置

### 腾讯云推荐配置

| 组件 | 规格 | 说明 |
|------|------|------|
| 应用服务器 | 4C8G × 2 | FastAPI服务 |
| 数据库 | PostgreSQL 16, 4C16G | RLS多租户 |
| Redis | 4G内存 | 缓存+消息 |
| 对象存储 | COS | 图片/文件 |
| CDN | 全国节点 | 前端静态资源 |

## Mac mini 首次部署

1. 安装 macOS + Homebrew
2. 安装 PostgreSQL 16 + Python 3.11+
3. 配置 Tailscale 连接云端
4. 运行初始化脚本（DB创建 + 数据同步）
5. 配置 launchd 服务（mac-station + coreml-bridge + sync-engine）
6. 连接 UPS + 配置断电自动关机
