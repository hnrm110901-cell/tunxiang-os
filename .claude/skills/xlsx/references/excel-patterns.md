# Excel 电子表格设计模式

## 餐饮行业常用公式

### 毛利率计算
```
毛利率 = (营业额 - 食材成本) / 营业额
= (B5 - C5) / B5
```

### 人效
```
人效 = 营业额 / 在岗人数
= B5 / D5
```

### 坪效
```
坪效 = 营业额 / 营业面积(㎡)
= B5 / E5
```

### 翻台率
```
翻台率 = 用餐桌次 / 总桌数
= F5 / G5
```

### 客单价
```
客单价 = 营业额 / 客流量
= B5 / H5
```

### 食材损耗率
```
损耗率 = (领用量 - 实际用量) / 领用量
= (I5 - J5) / I5
```

### 同比增长
```
同比增长 = (本期 - 同期) / 同期
= (B5 - K5) / K5
```

### 环比增长
```
环比增长 = (本期 - 上期) / 上期
= (B5 - L5) / L5
```

## 条件格式规则

### 毛利率预警（三条硬约束）
| 值范围 | 颜色 | 含义 |
|--------|------|------|
| < 30% | 红色底 (#F8D7DA) | 低于底线，严重预警 |
| 30%-40% | 黄色底 (#FFF3CD) | 接近底线，需关注 |
| > 40% | 绿色底 (#D4EDDA) | 健康 |

### 效期预警（三条硬约束）
| 值范围 | 颜色 | 含义 |
|--------|------|------|
| 已过期 | 红色底 + 加粗 | 禁止使用 |
| 3天内到期 | 黄色底 | 优先使用 |
| > 3天 | 无特殊格式 | 正常 |

### 出餐时间预警（三条硬约束）
| 值范围 | 颜色 | 含义 |
|--------|------|------|
| > 阈值 | 红色底 | 超时 |
| > 阈值×0.8 | 黄色底 | 接近超时 |
| ≤ 阈值×0.8 | 绿色底 | 正常 |

### 排名（Top/Bottom）
| 排名 | 样式 | 含义 |
|------|------|------|
| Top 3 | 金色加粗 | 表现最佳 |
| Bottom 3 | 红色 | 需改进 |

## 常用 Sheet 结构

### 门店日报
```
Sheet: 日报
  A: 日期
  B: 营业额
  C: 食材成本
  D: 毛利率 (=公式)
  E: 客流量
  F: 客单价 (=公式)
  G: 翻台率
  H: 在岗人数
  I: 人效 (=公式)
  J: 同比增长 (=公式)

最后一行: 合计/平均
```

### 库存盘点
```
Sheet: 盘点
  A: 分类
  B: 食材名称
  C: 单位
  D: 系统库存
  E: 实际库存
  F: 差异 (=E-D)
  G: 差异率 (=F/D)
  H: 保质期
  I: 到期日
  J: 备注

Sheet: 汇总
  按分类汇总差异
```

### 排班表
```
Sheet: 本周排班
  A: 员工姓名
  B: 岗位
  C-I: 周一至周日
  J: 本周总工时 (=SUM)

单元格内容: 早班/中班/晚班/休
颜色: 早=浅蓝, 中=浅黄, 晚=浅紫, 休=浅灰
```

### 菜品定价
```
Sheet: 定价表
  A: 菜品编码
  B: 菜品名称
  C: 分类
  D: 食材成本
  E: 建议售价 (=D/目标毛利率的倒数)
  F: 堂食价
  G: 外卖价
  H: 堂食毛利率 (=公式)
  I: 外卖毛利率 (=公式)
  J: 四象限分类 (明星/金牛/问号/瘦狗)
```

## 数据验证模板

### 下拉列表
```python
from openpyxl.worksheet.datavalidation import DataValidation

# 门店列表
store_dv = DataValidation(
    type="list",
    formula1='"尝在一起-总店,尝在一起-分店1,尝在一起-分店2,最黔线-总店"',
    allow_blank=False
)
store_dv.error = "请从下拉列表中选择门店"
ws.add_data_validation(store_dv)

# 班次
shift_dv = DataValidation(
    type="list",
    formula1='"早班,中班,晚班,休息"'
)
```

### 数值范围
```python
# 折扣率 0-100%
discount_dv = DataValidation(
    type="decimal",
    operator="between",
    formula1=0,
    formula2=1
)
discount_dv.error = "折扣率必须在0-100%之间"
```
