# 屯象OS 品牌视觉规范（文档版）

> 适用于所有输出文档（Word/PPT/PDF/Excel）

## 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主品牌色 | `#FF6B35` | 标题装饰线、重点标注、图表主色 |
| 深色辅助 | `#1E2A3A` | 表格表头背景、封面背景、正文标题 |
| 正文色 | `#333333` | 正文文字 |
| 次要文字 | `#666666` | 注释、脚注、次要信息 |
| 浅背景 | `#F8F9FA` | 表格交替行、引用块背景 |
| 成功/正向 | `#28A745` | 增长数据、达标指标 |
| 警告 | `#FFC107` | 需关注的数据 |
| 危险/负向 | `#DC3545` | 下降数据、违规、低于阈值 |

| 层级 | Word/PDF | PPT |
|------|----------|-----|
| 大标题(H0) | 28pt | 44pt |
| 章标题(H1) | 24pt | 36pt |
| 节标题(H2) | 18pt | 28pt |
| 小标题(H3) | 14pt | 22pt |
| 正文 | 11pt | 18pt |
| 注释/脚注 | 9pt | 12pt |

## 页面规范

### Word/PDF (A4)
- 页面：210 × 297mm
- 页边距：上下 2.54cm，左右 3.17cm
- 行距：1.5倍行距
- 段前段后：H1=24/12pt，H2=18/6pt，正文=6/6pt

### PPT (16:9)
- 幻灯片：13.333 × 7.5 英寸
- 安全区域：距边 0.5 英寸内不放关键内容
- 字间距：标题 0，正文 0

## 封面设计

### 标准封面元素
1. **公司标识** — "屯象科技" 或 Logo
2. **文档标题** — 大标题，居中
3. **副标题/文档类型** — 标题下方
4. **日期** — YYYY年MM月DD日
5. **机密等级** — "机密" / "内部" / "公开"
6. **版本号** — V1.0

### 封面配色方案
- **方案A（深色）：** TX_NAVY 背景 + 白色文字 + TX_PRIMARY 装饰线
- **方案B（浅色）：** 白色背景 + TX_NAVY 文字 + TX_PRIMARY 装饰色块

## 页眉页脚

### 页眉
- 左侧：`屯象科技 | 文档标题`
- 右侧：页码 `第X页/共Y页`
- 分隔线：TX_PRIMARY 色，0.5pt

### 页脚
- 居中：机密等级声明
- 字号：9pt，TX_TEXT_LIGHT 色

## 表格规范

- 表头：TX_NAVY 背景 + 白色粗体文字
- 数据行：交替 白色/TX_LIGHT_BG
- 边框：0.5pt，#D9D9D9
- 内边距：上下 6pt，左右 8pt
- 对齐：文字左对齐，数字右对齐，表头居中

## 图表规范

- 主色序列：TX_PRIMARY → TX_NAVY → #4ECDC4 → #45B7D1 → #96CEB4
- 网格线：#E9ECEF，0.5pt
- 标题：H3 字号，居中
- 图例：右侧或底部
- 数据标签：关键数据点标注数值
