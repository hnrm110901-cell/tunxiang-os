# 微信支付联调与上线检查清单

## A. 环境配置

- [ ] 配置 `WECHAT_PAY_MCH_ID`
- [ ] 配置 `WECHAT_PAY_API_KEY_V3`
- [ ] 配置 `WECHAT_PAY_CERT_PATH`
- [ ] 配置 `WECHAT_PAY_APPID`
- [ ] 回调域名已备案并可公网访问（HTTPS）

## B. 支付流程验收

- [ ] 创建预支付成功，返回 `timeStamp/nonceStr/package/signType/paySign`
- [ ] 小程序拉起支付成功
- [ ] 微信回调成功到达，签名验证通过
- [ ] 订单状态更新为已支付（且仅更新一次）
- [ ] 支付流水与订单金额一致

## C. 异常与幂等

- [ ] 重复回调不会重复入账
- [ ] 回调失败可重试，最终能收敛为正确状态
- [ ] 主动查单可补偿回调丢失场景
- [ ] 退款金额超限被正确拦截

## D. 监控告警

- [ ] 回调验签失败告警
- [ ] 支付成功但订单未完结告警
- [ ] 退款失败告警
- [ ] 日志字段脱敏（openid、证书、签名串）

## E. 回滚预案

- [ ] 预留 Mock/沙箱切换开关（仅非生产可用）
- [ ] 可快速降级到“查单确认后再更新状态”策略
- [ ] 记录版本与迁移步骤，支持快速回退
