## 三、步骤1：路由器配置

1. 设置路由器 SSID: `TunxiangOS-Demo`，密码: `tunxiang2024`
2. 设置 LAN 网段: `192.168.10.0/24`
3. 关闭 DHCP 或设置 DHCP 范围 `192.168.10.100 ~ 192.168.10.200`
4. Mac mini 和佳博打印机用网线连接路由器 LAN 口

## 四、步骤2：Mac mini M4 部署

### 前置条件
- macOS Sonoma 15+
- 安装 Docker Desktop for Mac（或 OrbStack，推荐OrbStack更轻量）
- 安装 Node.js 18+（通过 `brew install node`）
- 安装 pnpm 8+（通过 `npm install -g pnpm`）

### 固定IP
```bash
# macOS 系统偏好设置 → 网络 → 以太网 → 手动配置
# IP: 192.168.10.10
# 子网: 255.255.255.0
# 网关: 192.168.10.1
# DNS: 192.168.10.1, 223.5.5.5
```

### 一键部署
```bash
cd tunxiang-os
bash demo/demo-mac-setup.sh
```

部署完成后，Mac mini 上运行：
- PostgreSQL 16 (5432)
- Redis 7 (6379)
- Gateway API (8000)
- 13个业务微服务 (8001-8012)
- Nginx 反代 (80) — 静态资源 + API代理
- 前端构建产物（web-pos/web-admin/web-kds/web-crew）

## 五、步骤3：佳博打印机配置

1. 用网线连接佳博打印机到路由器
2. 佳博打印机默认IP: `192.168.123.100`，需修改为 `192.168.10.20`
3. 修改方法：
   - 打印自检页（长按FEED键开机），查看当前IP
   - 电脑浏览器访问打印机当前IP，进入Web管理界面
   - 修改IP为 `192.168.10.20`，子网 `255.255.255.0`，网关 `192.168.10.1`
   - 保存重启
4. 验证：`ping 192.168.10.20` 通即成功

## 六、步骤4：商米T2 配置

1. **连WiFi**: 连接 `TunxiangOS-Demo`
2. **固定IP**: 设置 → WLAN → 高级 → 静态IP → `192.168.10.30`
3. **安装WebView壳层**:
   - 将 `demo/sunmi-t2-config.json` 通过ADB推送到设备
   - 或直接用商米T2内置浏览器打开: `http://192.168.10.10/pos/`
4. **设为开机自启**: 商米T2设置 → 开机应用 → 选择浏览器
5. **打印机配置**: T2通过网络连接佳博打印机 `192.168.10.20:9100`

按F11全屏，或使用Chrome的"添加到主屏幕"功能。

## 十、常见问题

**Q: 商米设备连不上Mac mini？**
A: 确认在同一局域网，ping 192.168.10.10 能通。检查Mac mini防火墙设置（系统偏好→安全→防火墙→关闭或添加例外）。

**Q: 佳博打印机不打印？**
A: 检查IP是否设为192.168.10.20，确认端口9100开放。可用 `telnet 192.168.10.20 9100` 测试。

**Q: Docker服务启动慢？**
A: Mac mini M4首次拉取镜像约需5-10分钟，后续启动约30秒。建议提前一天部署完成。

**Q: 断网后能用吗？**
A: 可以。所有服务运行在Mac mini本地，局域网内完全自治。
