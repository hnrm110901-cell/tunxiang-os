## 一、V1 总体应用架构

7 个应用域，共享统一账号、权限、数据与 Agent 中台。

| 代号 | 应用域 | 面向角色 |
|------|--------|---------|
| A | Hub 总部经营台 | 老板、COO、区域总、品牌总、经营分析 |
| B | Front 前厅协同台 | 迎宾、预订员、楼面经理 |
| C | Store 门店执行台 | 店长、收银、服务员、值班经理、传菜 |
| D | Growth 会员增长台 | 会员运营、品牌营销、私域负责人 |
| E | Agent Studio / Ops | 产品、运营、实施、算法/平台、运维 |
| F | Platform 平台治理台 | 总部IT、实施、管理员、审计 |
| G | Mini 顾客侧 | 顾客、会员 |

## 二、一级/二级模块清单

### A. Hub 总部经营台

- **A1 经营驾驶舱**: 集团总览、品牌概览、区域概览、门店对比、预警中心、AI经营问答
- **A2 总部Agent决策中心**: 总控Agent工作台、经营分析Agent、预警处置Agent、任务编排、人机审批
- **A3 门店治理**: 门店档案、营业状态监控、巡店管理、整改闭环、日清日结汇总
- **A4 商品与菜单治理**: 菜单中心、品项中心、套餐与加价规则、时段菜单、售罄策略
- **A5 会员与增长总控**: 会员资产总览、人群分层、活动中心、券中心、触达分析
- **A6 数据与报表中心**: 日报、周复盘、月报、品类分析、菜品分析、翻台分析、人效分析
- **A7 组织与权限**: 组织架构、角色权限、数据权限、审计日志、消息订阅

### B. Front 前厅协同台

- **B1 迎宾工作台**: 今日预订概览、等位到店概览、当前桌态、AI迎宾建议
- **B2 预订管理**: 预订台账、新建预订、改约/取消、包厢预订、团餐/宴会预订
- **B3 等位管理**: 等位队列、叫号台、过号/取消、候位安抚建议
- **B4 桌台管理**: 桌态总览、桌区管理、并台/拆台、翻台预测、入座分配建议
- **B5 前厅Agent协同**: 预订Agent、等位Agent、桌台调度Agent、高价值客识别、客诉风险

### C. Store 门店执行台

- **C1 当班工作台**: 班前简报、今日任务、营业状态、当班异常、AI当班助手
- **C2 点单与服务**: 开台、点单、加菜/退菜、催菜/起菜、服务记录、服务提醒
- **C3 收银与结账**: 待结订单、合单/拆单、收银台、部分支付、退款/冲正、发票
- **C4 厨房协同**: KDS总览、工位队列、出品进度、异常挂起、缺货/停售联动
- **C5 店长运营**: 午市复盘、晚市复盘、日清日结、差异解释、整改任务、人效追踪
- **C6 门店消息中心**: 系统消息、预警消息、Agent建议、审批待办、任务通知

### D. Growth 会员增长台

- **D1 会员资产**: 会员总览、会员详情、标签、等级、成长旅程
- **D2 人群与分层**: 活跃/沉睡/高价值/新客/自定义人群包
- **D3 活动与券**: 活动中心、券模板、发券任务、核销分析、ROI
- **D4 触达编排**: AI触达助手、触达任务、文案生成、发送记录、效果归因
- **D5 会员Agent中心**: 会员增长Agent、召回Agent、复购Agent、节日活动Agent

### E. Agent Studio / Ops

- **E1 Agent目录**: 列表、详情、版本管理、发布记录
- **E2 Prompt/Policy管理**: 系统Prompt、业务策略、安全边界、输出模版、版本对比
- **E3 Tool/MCP工具中心**: 预订/等位/桌台/订单/支付/会员/报表/消息工具
- **E4 Workflow编排**: 工作流列表、节点编排、条件分支、人工审批、回退补偿
- **E5 Knowledge/Memory**: SOP知识库、菜品/会员/门店知识库、会话记忆、业务记忆
- **E6 监控与评测**: 对话日志、工具调用、成功率、延迟、成本、错例库

### F. Platform 平台治理台

- **F1 门店与设备**: 门店清单、Edge节点、POS/KDS/打印机管理、设备告警
- **F2 集成与接口**: 支付通道、第三方、Webhook、API Key、回调日志
- **F3 配置中心**: 菜单/价格/营业日历/时段规则/组织/环境配置
- **F4 风控与审计**: 操作日志、审批日志、风险规则、异常事件
- **F5 权限中心**: 用户、角色、岗位、门店范围、数据域授权

### G. Mini 顾客侧

- **G1 顾客预订**: 选门店、选时段、提交、确认
- **G2 顾客等位**: 线上排号、进度、到号提醒、过号
- **G3 顾客点单**: 扫码入桌、菜单浏览、下单、加菜、订单进度
- **G4 顾客结账**: 账单查看、自助支付、支付结果、发票
- **G5 会员中心**: 会员首页、积分/券、等级、消费记录、活动权益

## 五、关键页面联动规则

1. **预警卡片联动**: `/hub/alerts` → `/hub/stores/:storeId/status` + 自动打开Agent分析抽屉
2. **预订到桌台**: `/front/reservations/:id` → `/front/tables` + 高亮推荐桌位
3. **等位到入座**: `/front/waitlist/:id` → `/front/tables/:tableId` + 一键开台
4. **报表到整改**: `/hub/reports/weekly` → `/hub/rectifications/:id` + 自动生成整改草稿
5. **Agent到审批**: `/agent/workflows/:id/debug` → `/hub/approvals` 或 `/store/messages/approvals`
