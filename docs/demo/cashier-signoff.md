## 签字要求

每位收银员独立完成以下 5 个场景，每个场景计时 ≤ 2 分钟视为通过：

- [ ] **场景 1：扫码点餐 + 现金支付**（单桌 3 道菜 + 现金找零）
- [ ] **场景 2：会员折扣 + 优惠券抵扣**（识别 VIP + 叠加规则）
- [ ] **场景 3：退菜 + 改单**（高峰期突发退菜）
- [ ] **场景 4：日结 + 小票补打**（当日收银表核对）
- [ ] **场景 5：外卖订单接单**（美团推送 → 后厨 → 打包）
