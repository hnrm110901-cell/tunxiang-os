## 六、安全审计关键发现（2026-03-27）

### P0 紧急

- **CRITICAL-001**：BOM/waste_events 表 RLS 使用错误 session 变量 → RLS 无效
- **CRITICAL-002**：RLS NULL 绕过漏洞（`OR ... IS NULL` 允许无租户上下文访问）

### P1 高危

- ORM tenant_filter 仅覆盖 18 张表（11% 覆盖率，共 162 model 文件）
- Nginx 缺 rate limiting
- X-Tenant-ID header 信任问题
- 生产 docker-compose 暴露内部端口

### P2 中危

- 32 个 model 缺少 tenant_id/store_id 字段
- Nginx 缺 CSP/Permissions-Policy header
- disable_tenant_filter 无权限检查
