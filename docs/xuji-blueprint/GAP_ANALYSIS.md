## 各文档关键差距

### 1. 架构 V2（1内核 + 3中台 + 6前台域）
- **对齐**: 五层架构、8域微服务、6大实体
- **差距**: "日清日结操作层(E1-E8)" 完全缺失；Agent 三分类(守门员/优化/增长)需与现有9 Skill Agent映射

### 2. 信息架构 IA + 导航树
- **对齐**: ShellHQ 四栏布局 + SidebarHQ 菜单分组
- **差距**: 6 个经营中心 × 48 二级菜单未完整实现；角色化菜单裁剪仅后端(menu_config.py)有逻辑，前端未接入

### 3. 页面清单 + 路由规划
- **对齐**: 5 个前端应用结构一致
- **差距**:
  - web-pos: 5/12 P0页面 (缺预订/异常/设置等)
  - web-kds: 1/5 路由 (仅看板)
  - web-crew: 3/6 路由 (缺日清日结)
  - web-admin: 3/10 一级模块

### 4. P0 核心页面原型
- **对齐**: 桌台总览 ✓、收银结账 ✓、KDS看板(基础) ✓
- **差距**: 缺开台点单三栏布局、日清日结巡航、店长复盘页

### 5. 字段明细(200+ 字段)
- **对齐**: 基础审计字段(tenant_id/timestamps/is_deleted)、金额存分
- **差距**: dining_duration_min/abnormal_flag/margin_alert_flag 等业务字段未入模型

### 6. 3年路线图
- **对齐**: Year 1 徐记优先 + 折扣守护 Agent
- **差距**: 行业模板机制(Pro/Standard/Lite)未建立
