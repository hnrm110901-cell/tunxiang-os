### 网络安全要求

1. **VLAN隔离**: 收银网络与客用WiFi必须在不同VLAN
2. **收银网不允许外网访问**: 收银设备仅访问Mac mini和云端API
3. **Mac mini通过Tailscale VPN连接云端**: 加密隧道，不暴露公网端口
4. **WiFi密码定期更换**: 客用WiFi每月更换密码

## 四、通信协议说明

### ESC/POS (热敏打印机)
- **连接方式**: TCP Socket, 端口 9100
- **编码**: GBK (中文)
- **适用设备**: 所有热敏/针式票据打印机
- **已实现**: `services/tx-trade/src/services/printer_driver.py`

### 串口协议 (电子秤)
- **连接方式**: RS232 / USB虚拟串口
- **参数**: 9600bps, 8N1 (8数据位, 无校验, 1停止位)
- **适用设备**: 顶尖/大华/凯士/梅特勒-托利多电子秤
- **数据格式**: 品牌各异，已在 `protocol_support.py` 中做适配

### HID 键盘模拟 (扫码枪)
- **连接方式**: USB HID
- **模式**: 键盘模拟 (默认)
- **适用设备**: 新大陆/霍尼韦尔/斑马扫码设备
- **数据格式**: 以回车符(CR/LF)结尾的字符串

### ZPL/TSPL (标签打印机)
- **连接方式**: TCP Socket 或 USB
- **ZPL**: 斑马(Zebra)标签打印机专用指令集
- **TSPL**: TSC/得力/佳博标签打印机通用指令集

### 商米 JS Bridge
- **连接方式**: WebView JavaScript 接口
- **适用设备**: 商米T2/T2s/V2 Pro等安卓POS
- **接口**: `window.TXBridge.*`
- **已定义**: `apps/android-shell/` 壳层

### HTTP API (智能设备)
- **连接方式**: HTTP REST
- **适用设备**: 排队叫号机/自助点餐机/KDS
- **格式**: JSON

## 六、故障排查

### 打印机不出票
1. 检查网线/WiFi连接是否正常
2. ping 打印机 IP 确认网络通
3. telnet 打印机 IP 9100 确认端口通
4. 检查纸卷是否装好
5. 使用 `/store/{store_id}/test` 接口测试

### 电子秤无数据
1. 检查 USB/串口线是否接好
2. 确认串口参数(波特率/数据位/校验位)匹配
3. 使用串口调试工具查看原始数据

### 扫码枪无反应
1. 确认 USB 连接正常
2. 检查扫码枪是否处于 HID 键盘模拟模式
3. 用记事本测试扫码是否有输出

### 钱箱不弹出
1. 确认 RJ11 线缆连接到打印机钱箱接口
2. 确认打印机在线
3. 手动发送 ESC p 指令测试
