## 一、产品概述

G10云仓（Bestech Software）是一款面向连锁餐饮行业的综合管理软件，涵盖组织架构管理、商品管理、供应商管理、价格管理、POS前端管理、财务管理、客户管理、应收应付管理等多个业务领域，为餐饮企业提供从供应链到门店运营的全链路数字化管理能力。

### 核心特性
- **多商户/多品牌支持**：支持集团化多商户、多品牌、多区域运营
- **全供应链覆盖**：从供应商管理、采购定价到仓储配送一体化
- **灵活定价体系**：支持采购定价、销售定价、零售定价多种定价模式
- **POS前端集成**：云POS菜谱管理、设备打印、线上线下餐厅全覆盖
- **财务核算**：科目管理、租金管理、应收应付、工费分摊
- **工作流引擎**：可配置的审批流程管理

## 四、系统架构特点

### 4.1 技术架构
- **前端框架**：基于 jQuery + Bootstrap 的传统 Web 应用
- **后端架构**：ASP.NET MVC（基于 URL 路径推断）
- **数据可视化**：集成 ECharts 图表库
- **富文本编辑**：集成 TinyMCE 编辑器
- **静态资源服务**：独立静态资源服务器（端口 7002）

### 4.2 系统特性
- **多租户架构**：基于商户编号隔离的多租户设计
- **iframe 嵌套**：主框架 + iframe 内容区布局模式
- **基于角色的权限控制**：支持菜单级、操作级权限控制
- **工作流引擎**：可配置的审批流程管理
- **多平台对接**：支持美团、饿了么等外卖平台对接

### 4.3 登录认证
- **商户编号**：系统级租户标识
- **用户编号**：手机号作为用户名
- **密码**：用户密码
- **验证码**：数字验证码 + 图片滑块验证双重验证
- **登录方式**：支持密码登录和手机验证码登录
