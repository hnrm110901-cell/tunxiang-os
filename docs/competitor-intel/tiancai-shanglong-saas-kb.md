### 版本策略

- 增值功能需额外付费(智慧商街/自助点餐机/AI/MAC绑定等)

## 二、POS门店功能 (~370项)

### 核心流程
- 开班(备用金) → 客位管理(开/换/并/关/搭台/废单/团队) → 点单(规格/做法/半份/临时/拼菜/套餐) → 结算(20+活动方案 + 20+支付方式) → 结班/日结
- 外卖(美团/饿了么/抖音自动接单+厨打) + 预订(日历/预订单/来电) + 存酒(增值) + 押金

### 结算方式
人民币/银行卡/会员卡(CRM6/7)/微信/支付宝/闪惠/银联/代金券/团购/挂账/支票/跑单/宴请/预订押金

### 活动方案
比例优惠/类别优惠/单品优惠/全单优惠/手工定额/品项打折/账单满减/第N杯打折/定额优惠/优惠券/品项满减

## 三、KDS智能出品 (8种模式, ~136项)

1. **标准配菜** — 按菜品/客位显示, 超时/催单/赠单变色, 做法筛选/组合筛选
2. **标准传菜** — 扫码传菜, 催菜, 传菜准备
3. **按单配菜** — 企业版, 单个/整单完成, 新订单语音提示
4. **按单传菜** — 企业版, 先上餐具, 自提开关, 订单核销
5. **外卖模式** — 企业版
6. **计件配菜** — 企业版, 选择厨师, 多键盘操作
7. **备餐模式** — 企业版, 选厨师+数量+刷卡
8. **厨房控菜** — 格子大小可调, 客位绑定, 品项设置, 自动翻页
- 海鲜称重(去皮/清零) + 泳道传配(企业版) + 智能分配(上下班/人工)

## 六、食尚订 (客源聚合预订)

### 核心功能
- **客源聚合**: 美团点评/百度/高德/支付宝 + 来电/亲访/微信/小程序
- **终端**: PC预订台 + 手机APP + 智能预订电话Pro(来电弹窗/通话录音/核餐回拨) + 平板 + 小程序
- **预订管理**: 预点菜品 + CRM会员同步 + 集团统一预订/监控
- **宴会全流程**: 商机→合同→EO工单→准备→执行→回访→二次营销
- **销售CRM**: 目标管理(年/月/人) + 任务管理(8种任务类型) + 客户资料完整度考核
- **运营数据**: 客户分析(活跃/沉睡/流失) + 订单分析(渠道/餐段/类型) + 翻台率/上座率 + 宴会转化率

## 七、CRM会员+智慧餐厅

- **排队**: 关注公众号(沉淀粉丝) + 会员标签识别 + 超时返券 + 排队预点菜
- **扫码点餐**: 微信/支付宝/云闪付 + 多人同时 + 多套模板 + 做法加价 + 锅底先下单 + 分时段 + 智能推荐
- **桌边付**: 会员余额+积分+券核销 + 互斥共享
- **智能储值**: 按当餐客单价自动计算(充2倍7折/充3倍5折/充5倍免单)
- **营销20+**: 裂变砍价/三级分销/消费返现/N次赠/买A赠B/券包/错时/会员日/升降级
- **积分商城+甄选商城**: 实物/券兑换 + 餐饮电商(原材料/半成品/衍生品)

## 八、云供应链 SCM (5级)

1. **单店基础**: 进销存 + POS对接 + 成本卡
2. **单店升级**: 供货商EDI + 智能补货 + 采购管理
3. **单店扩展**: 应收应付
4. **连锁标准**: 店间调拨(可加价)
5. **连锁升级**: 中心库存 + 连锁配送(3种发货) + 生产加工(BOM) + MRP预估 + 外销管理

运营指标: 分差±2% / 库存周转1.5-2天 / 无成本卡比例0.5%
