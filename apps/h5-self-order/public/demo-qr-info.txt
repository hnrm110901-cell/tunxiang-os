演示二维码信息
==============

门店：尝在一起·文化城店
桌号：A03

演示 URL（手机扫码或直接打开）：
https://demo-m.tunxiangos.com/m?store=czyz-wh001&table=A03&demo=true

本地开发调试 URL：
http://localhost:5181/m?store=czyz-wh001&table=A03&demo=true

完整演示流程：
1. 扫码 / 打开 URL → 入口动画页（显示门店名+桌号，2秒后自动跳转）
2. 菜单页 → 左侧湘菜分类，右侧菜品（剁椒鱼头/口味虾/毛氏红烧肉等）
3. 加菜 → 底部购物车悬浮栏实时更新
4. 购物车页 → 可调整数量/删除/填写备注
5. 结算页 → 选择支付方式（微信支付/支付宝/现场结账）
6. 点击任意支付方式 → 3 秒 loading → 支付成功页
7. 成功页 → 显示订单号 + 出餐进度 + 继续点餐按钮

注意：demo=true 参数触发演示模式，无需真实支付，点击支付后 3 秒自动跳转成功页。
